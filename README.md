## Getting Started

The Poll Project runs locally. 

### Prerequisites

To run the Poll Project, you must have:
* npm
  ```sh
  npm install npm@latest -g
  npm install . 
  ```
* node.js: https://nodejs.org/en/download
* Visual Studio Code
* SQL Server Management Studio
* TailwindCSS
  ```sh
  npm install -D tailwindcss
  npx tailwindcss init
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JesusFranC/CECS443-Team3-spring24.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start a new terminal and run:
  ```sh
  cd client
  npm start
  ```
4. Start another terminal and run:
   ```sh
   cd server
   dotnet run
   ```
